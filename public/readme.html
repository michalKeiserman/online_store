what additional paged did we add?
  • Contact: press on the contact us button in the menu -> write a message to the website owners in the form and press the send button. then, the message will show up in the messages page.

  • Messages: available only for the admin. press the messages button that is located in the admin page -> see all messages sent from users.

  • Gift card: press on the giftcard button in the menu -> write the amount of dollars you wish to send and the name of the user you want to send it to it the form. then, press the send button, and the cash will be updated in the receiving user wallet property.

  • Account: press on the account us button in the menu -> see the user name that is connected now, his accumulated money in his wallet, and change password if required - write the new password in the input text and press submit. then, it will be updated in the user password property.

  • Top 5: press on the top 5 button in the menu -> see the top 5 products (you can add them to your cart if you wish).

  • Recommendation: press on the recommendation button in the menu -> see recommendation.
